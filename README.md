## 📌 Project Overview
AgroSmart AI adalah sistem cerdas yang dirancang untuk membantu peternak dalam mendeteksi penyakit ternak secara dini menggunakan data sensor IoT. Proyek ini mengintegrasikan data fisiologis sapi untuk memberikan prediksi kesehatan yang akurat.

## 🎯 Key Objectives
- **Early Detection:** Mengurangi risiko penyebaran penyakit antar ternak.
- **Precision Farming:** Optimalisasi manajemen kesehatan sapi berbasis data.
- **Predictive Analytics:** Menggunakan Machine Learning untuk klasifikasi status kesehatan.

## 🛠️ Tech Stack
- Python (Pandas, Seaborn, Scikit-Learn)
- Jupyter Notebook
- Machine Learning (Classification Pipeline)
